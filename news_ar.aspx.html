الخدمات الالكترونية

في إطار سعي شركة عبد اللطيف جميل المتحدة للتمويل في تقديم خدمات متميزة للضيف، يسر شركة عبد اللطيف جميل للتمويل الإعلان عن إضافة خدمات ما بعد البيع الإلكترونية.

تتيح هذه الخدمات للضيف القيام بمعظم إجراءات ما بعد البيع من خلال الإنترنت بكل طمأنينة وراحة في أي وقت دون الحاجة لزيارة الفرع.

معالجة طلبك أصبحت أسهل وأسرع من خلال حسابك الخاص عبر موقع الشركة. أدناه قائمة الخدمات الإلكترونية المتوفرة وبإمكانك الضغط على اسم الخدمة للإطلاع على طريقة الاستفادة منها:

  • تجديد الاستمارة. ( إضغط هنا )
  • نقل ملكية السيارة. ( إضغط هنا )
  • شراء السيارة. ( إضغط هنا )
  • إضافة سائق إضافي. ( إضغط هنا )
  • تجديد السائق الإضافي. ( إضغط هنا )
  • حذف السائق الإضافي. ( إضغط هنا )
  • طباعة تفويض القيادة. ( إضغط هنا )
  • استعادة المركبة. ( إضغط هنا )
  • طباعة وثيقة التأمين. ( إضغط هنا )
  • طباعة خطاب إخلاء الطرف. ( إضغط هنا )
